# Human-Disease-Detection
A beginner-friendly Human Disease Detection web project built using HTML, CSS, and JavaScript. It allows users to enter symptoms and get possible disease predictions through a simple interactive interface. The project focuses on basic frontend development, user input handling, and creating an easy-to-use healthcare-themed web application.

---

## Features
- User-friendly interface
- Symptom-based disease prediction
- Responsive web design
- Interactive functionality using JavaScript
- Easy to understand project structure

---

## Technologies Used
- HTML
- CSS
- JavaScript
